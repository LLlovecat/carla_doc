发布可执行程序

  • make launch时报错：
could not find any instance of Visual Studio.
  1. 终止BuildLibCarla.bat的执行可以开始编译虚幻编辑器。

  2. 注释掉Util\BuildTools\BuildCarlaUE4.bat中“构建 Carla 编辑器”的部分。

  3. 启动虚幻编辑器时报错：

由于找不到 XINPUT1_3.dll，无法继续执行代码。

该动态库文件位于engine\Engine\Binaries\ThirdParty\AppLocalDependencies\Win64\DirectX目录下，应该是虚幻引擎编译时向系统注入了一些动态库文件。

使用命令行编译工程sln

"C:\\Program Files (x86)\\Microsoft Visual Studio\\2019\\Enterprise\\Common7\\IDE\\\devenv.com" UE4.sln /Build
